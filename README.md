# AI Based Smart Attendance System

This project is an AI-powered facial recognition attendance system designed to automatically mark attendance using CCTV or camera input.

## Features
- Automatic face detection
- Facial recognition for student identification
- Subject-wise attendance tracking
- Daily attendance reports

## Tech Stack
- Python
- OpenCV
- Face Recognition Library
- NumPy
- Pandas

## How it Works
1. Camera captures classroom images
2. System detects faces
3. Matches faces with student database
4. Marks attendance automatically
5. Saves attendance in CSV file
